This is the easy to use D-Net GUI application.

Usage instructions:
	1. Ensure you update the camera parameters in the config.py file global variables.
	2a. To run the file from terminal, use: python3 viewer.py 
	2b. If you a using an IDE such as Spyder, you can simply run the file viewer.py

Button functions:
	- Select the Depth button to view the colour coded depth output.
	- Select the Contours button to view the colour coded depth output with contour overlays.
	- Select home to go to the home screen.
	- Select settings to select a different model variation or to adjust the number of contours displayed in Contours mode.
